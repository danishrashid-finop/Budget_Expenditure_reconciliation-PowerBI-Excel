Department Budget Utilisation Dashboard (Power BI)

Objective
To design an interactive Power BI dashboard that enables finance teams to monitor department wise payments against allocated budgets and track utilisation trends.

Business Context
Finance departments require real time visibility into spending patterns to prevent budget overruns and support audit readiness. This dashboard simulates a monthly financial monitoring system used in large organisations.

Data Sources
Excel based payments data (transaction-level)
Department master data
Monthly budget allocation data

Key Features
Department wise total payments
Budget vs actual comparison
Budget utilisation percentage
Interactive slicers for department and month
Clean, audit friendly layout

Tools Used
Microsoft Excel (data preparation)
Power BI Desktop (data modeling, DAX, visuals)

Outcome :
The dashboard allows stakeholders to quickly identify:

High spending departments
Under utilised budgets
Monthly spending trends# Budget_Expenditure_reconciliation PowerBI, Excel
